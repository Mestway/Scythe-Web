Synthesizing Highly Expressive SQL Queries

View the Project on GitHub (Mestway/Scythe)

Scythe is a programming by example system that aims to help end-users with database querying tasks. Given a pair of input-output table examples, Scythe synthesizes queries that can turn input examples into provided output examples.

When a user want to query a database but don't know how ...

... the user would go to Stack Overflow and write a post and wait for experts...

Now, use Scythe!

and the query can be obtained in ease!
